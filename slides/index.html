Architecture


   [React 19 + Vite]                            [GitHub Actions CI]
        │  HTTPS / JSON                         │  pytest, bandit, pip-audit
        ▼                                       ▼
   [Flask API Gateway]  ────────────────► [PostgreSQL on Railway]
        │
        ├─ WebScraper        ─► (requests + BeautifulSoup + lxml)
        ├─ SEOAnalyzer       ─► (9 indicateurs scorés)
        ├─ ContentAnalyzer   ─► (lexical + langdetect)
        ├─ AIAnalyzer        ─► [OpenAI GPT-4]  ◄─ Cache 24h
        │                                          [Redis Upstash]
        └─ ReportGenerator   ─► (fpdf2 PDF, JSON)

Monolithe Python, MVC strict, modules de service découplés.
